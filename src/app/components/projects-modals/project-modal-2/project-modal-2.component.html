Programmation Graphique Interactive et Animation pour le Web

Ce projet universitaire consistait en la création d'une plateforme interactive pour l'apprentissage de JavaScript et Canvas 2D. La plateforme regroupe plusieurs exercices pratiques qui offrent aux utilisateurs une expérience d’apprentissage immersive et dynamique.

  • Type: Plateforme Web Interactive
  • Technologies: JavaScript, HTML5, CSS3, PHP
  • Platform: Développement personnalisé, Canvas 2D
