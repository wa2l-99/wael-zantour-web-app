GFlotte – Application Web de Gestion de Flotte Automobile

Ce projet, réalisé dans le cadre de mon stage de fin d’études, vise à développer une application web centralisée pour la gestion de flotte automobile. L'objectif principal était d'automatiser la gestion des véhicules, des conducteurs, des missions ainsi que des opérations d'entretien et de maintenance, tout en optimisant les coûts opérationnels liés à la consommation de carburant et aux entretiens

  • Type: Application Web de Gestion de Flotte
  • Technologies: JavaScript (Node.js, React.js), HTML5, CSS3
  • Platform: Architecture MVC, Dashboard
